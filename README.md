# shan-cv

Personal CV site for Asam Shan — Senior Full Stack Engineer (Web3 Product).

Built with a cyberpunk terminal aesthetic: deep charcoal base, electric cyan-mint accent, matrix rain background, and staggered boot-sequence animations.

## Tech Stack

- **Next.js** (App Router)
- **TypeScript** (strict mode)
- **Tailwind CSS** v4
- **Lucide** icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## CV Downloads

PDF and DOCX versions are available in `public/` and downloadable directly from the site.

## Deployment

Standard Next.js deployment — works with Vercel, Netlify, or any Node.js host.

```bash
npm run build
npm start
```
